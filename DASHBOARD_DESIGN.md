## 🎯 핵심 기능 상세

### 1️⃣ **KPI 카드** (Metrics Cards)

#### Zone별 실시간 현황
```python
┌──────────────────────────┐
│ 🌡️ Zone 1 - 현재 온도     │
│                          │
│     31.2°C               │
│     ↑ +0.3°C (10분간)    │
│                          │
│ 범위: 29.5 ~ 32.1°C      │
│ 업데이트: 14:30          │
└──────────────────────────┘
```

#### 30분 후 예측
```python
┌──────────────────────────┐
│ 🔮 30분 후 예측           │
│                          │
│     31.8°C               │
│     ↑ +0.6°C 상승 예상   │
│                          │
│ 신뢰도: 95%              │
│ 예측 시각: 15:00         │
└──────────────────────────┘
```

#### 모델 정확도
```python
┌──────────────────────────┐
│ 🎯 예측 정확도            │
│                          │
│ MAE:  0.42°C             │
│ RMSE: 0.58°C             │
│ MAPE: 1.3%               │
│                          │
│ 평가 기간: 최근 24시간   │
└──────────────────────────┘
```

#### 경고 상태
```python
┌──────────────────────────┐
│ 🚨 경고 상태              │
│                          │
│  ✅ 정상                 │
│                          │
│  활성 경고: 0건          │
│  오늘 경고: 2건 (해결됨)│
└──────────────────────────┘
```

### 2️⃣ **실시간 알림 배너**

#### 경고 예시
```
⚠️ [15:00] Zone 3: 30분 후 32.5°C 예상 (임계값 32.0°C 초과 예정)
   → 냉각 시스템 점검 권장

🔥 [14:45] Zone 2: 급격한 온도 상승 감지 (+1.2°C/15분)
   → 서버 부하 확인 필요

✅ [14:30] 모든 존 정상 범위 유지 중
```

---

### 3️⃣ **메인 차트**

#### 실제 vs 예측 (Interactive Line Chart)
- **X축**: 시간 (zoom/pan 가능)
- **Y축**: 온도
- **라인**:
  - 실제 온도 (파란색 실선)
  - 예측 온도 (주황색 점선)
  - 신뢰 구간 (음영 영역)
- **호버**:
  - 시각: 2025-09-01 14:30
  - 실제: 31.2°C
  - 예측: 31.5°C
  - 오차: +0.3°C

#### 예측 오차 분석 (Error Bar Chart)
- 시간대별 예측 오차 분포
- 임계값 라인 표시
- 색상: 정상(녹색), 주의(노란색), 위험(빨간색)

---

### 4️⃣ **존별 비교**

#### 현재 온도 비교 (Horizontal Bar Chart)
```
Zone 1  ████████████████░ 31.2°C
Zone 2  ██████████████░░░ 30.8°C
Zone 3  ████████████████░ 31.5°C
Zone 4  ████████████████░ 31.3°C
        ├────────────────┤
       29°C    30°C    31°C    32°C
                       ↑ 임계값
```

#### 30분 후 예측 비교
- 예측 온도 + 신뢰 구간
- 상승/하락 추세 화살표

## 🎨 색상 팔레트

```python
# 온도 단계별 색상
TEMP_COLORS = {
    'safe':     '#4CAF50',  # 녹색 (< 30°C)
    'normal':   '#2196F3',  # 파란색 (30-31°C)
    'warm':     '#FF9800',  # 주황색 (31-32°C)
    'hot':      '#FF5722',  # 빨간색 (> 32°C)
}

# 예측 vs 실제
LINE_COLORS = {
    'actual':    '#1f77b4',  # 파란색
    'predicted': '#ff7f0e',  # 주황색
    'confidence':'#d3d3d3',  # 회색 (신뢰구간)
}

# 경고 레벨
ALERT_COLORS = {
    'success': '#d4edda',  # 연한 녹색
    'warning': '#fff3cd',  # 연한 노란색
    'danger':  '#f8d7da',  # 연한 빨간색
}
```

## 📱 반응형 디자인

### Desktop (> 1200px)
- 4열 KPI 카드
- 2열 차트 배치

### Tablet (768px ~ 1200px)
- 2열 KPI 카드
- 2열 차트 유지

### Mobile (< 768px)
- 1열 세로 배치
- 스크롤 가능
